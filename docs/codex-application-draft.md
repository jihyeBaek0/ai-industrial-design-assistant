## Short Project Description

AI Industrial Design Assistant is an open-source Flask web application that helps industrial design students, product designers, and design competition participants generate structured design project documents from a product idea or competition topic.

The app currently supports placeholder-based generation, project type selection, responsive card-based results, Markdown export, documentation, automated tests, and GitHub Actions CI.

## Why This Project Matters

Industrial design projects require more than visual ideas. Designers need to organize research, define target users, identify pain points, create personas, explore CMF direction, develop product concepts, prepare rendering prompts, and write presentation scripts.

Many AI tools generate generic text, but this project focuses specifically on the industrial design workflow.

The goal is to build a practical open-source assistant that helps students and designers structure the early stages of real design projects.
