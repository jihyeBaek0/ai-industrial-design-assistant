# Design Workflow

AI Industrial Design Assistant is designed around the early-stage industrial design process.

The goal is not only to generate text, but to help users organize the thinking process behind a product design project.

## Supported Workflow

The current version supports the following workflow:

1. Product idea or competition topic input

2. Design brief generation

3. Target user definition

4. User pain point exploration

5. Persona creation

6. Market research summary

7. SWOT analysis

8. Design keyword exploration

9. CMF direction

10. Product concept development

11. Rendering prompt generation

12. Presentation script preparation

13. Markdown export

## 1. Product Idea or Competition Topic

The user begins by entering a product idea, design problem, or competition topic.

Example:

```text

A modular desk organizer for industrial design students
