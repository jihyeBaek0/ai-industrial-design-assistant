# Project Overview

## AI Industrial Design Assistant

AI Industrial Design Assistant is an open-source web application designed to support the early stages of industrial design and product design projects.

The tool helps users transform a simple product idea or design competition topic into a structured set of design documents, including a design brief, target user, user pain points, persona, market research, SWOT analysis, design keywords, CMF suggestions, product concept, rendering prompt, and presentation script.

## Problem

Industrial design students and product designers often need to create many connected documents before they can begin sketching, modeling, rendering, or presenting a project.

This early phase can be difficult because designers need to connect multiple types of thinking:

- User research

- Market understanding

- Product positioning

- Visual direction

- Material and finish decisions

- Concept development

- Presentation storytelling

Many AI tools can generate text, but they are often too general. This project focuses specifically on the industrial design workflow.

## Target Users

This project is designed for:

- Industrial design students

- Product design students

- Designers preparing for design competitions

- Early-stage product designers

- Makers and creators developing product ideas

- Design educators who want to demonstrate structured design thinking

## Current Version

Version 1 uses placeholder data instead of connecting directly to the OpenAI API.

This decision was intentional because the first goal is to build a stable open-source project foundation:

- Clear Flask project structure

- Working frontend and backend connection

- Responsive user interface

- Card-based result layout

- Public GitHub repository

- Basic documentation

- Repeatable local setup

The OpenAI API integration will be added in a later version.

## Design Workflow Supported

The application currently supports the following design document sections:

1. Design Brief

2. Target User

3. User Pain Points

4. Persona

5. Market Research

6. SWOT Analysis

7. Design Keywords

8. CMF Suggestions

9. Product Concept

10. Rendering Prompt

11. Presentation Script

## Why This Project Matters

Industrial design is not only about making objects look good. A strong design project needs a clear reason, a defined user, a meaningful problem, material direction, market awareness, and a strong presentation story.

This project aims to help designers organize those elements faster, especially in the early concept development phase.

## Project Philosophy

This project is built around three principles:

### 1. Practical for real design work

The output should help designers begin an actual project, not only generate generic text.

### 2. Beginner-friendly and open-source

The project should be easy to run, understand, modify, and improve.

### 3. Human-centered design support

The tool should support the designer’s thinking process rather than replace it.
